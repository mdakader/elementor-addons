Easy Addons For Elementor is a plugin for widgets to extend for Elementor Page Builder.

== Description ==

Easy Addons is a plugin for **[Elementor](https://wordpress.org/plugins/elementor/)** page building. It will add some widgets that will give you the flexibility to design beautiful sections for your Elementor based websites.

**[View Demo](https://babuwp.com/wp/easy-addons/)**

= Available Widgets =

<ul>
    <li>Dual Heading</li>
    <li>Flip Box</li>
    <li>Call To Action</li>
    <li>Service Card</li>
    <li>Image Gallery</li>
    <li>Team</li>
    <li>Testimonial</li>
    <li>Price Table</li>
    <li>Post Card</li>
    <li>Counter Up</li>
</ul>


== Installation ==

1. Install Easy Addons either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2. After activating Easy Addons, navigate to the Elementor Addons.
